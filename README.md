# 深圳国际交流书院介绍

## 项目名称与一句话简介

“深圳国际交流书院介绍”是一座以真实校园资料讲述学校历史、人文、生活与全球连接的全中文互动网站。

## 在线访问

<https://dariohan.github.io/scie-introduction/>

## 选题方向

B

## 主要功能（200 字以内）

网站设置学校概况、地标与风景、人文与特色、美食与生活、未来寄语五个章节，包含互动时间轴、动画数据、校园图纸、照片画廊、视频、人物故事、课表与学生日程。终章通过滚动控制世界连线与校徽融合动画，并针对电脑、平板和手机进行响应式适配。

## 启动方式

1. 双击本文件夹根目录中的 `启动网站.command`。
2. 终端显示“网站已启动”后，浏览器会自动打开网站。
3. 使用结束后，关闭该终端窗口即可停止网站。

启动环境要求：电脑已安装 Node.js 18 或 Python 3（满足任意一项即可）。启动器会自动选择可用方式；启动静态成品不需要安装依赖，也不需要联网。

## 提交内容

- `网站成品/`：运行 `npm run build:static` 后生成的完整静态成品；桌面提交包已包含，公开仓库不重复跟踪生成文件
- `app/`、`components/`、`lib/`、`public/` 等：项目源代码与原始网页资源
- `README.md`：项目与启动说明
- `复盘文档.md`：任务复盘，仅保留在桌面提交包中，避免公开协作截图

## 开发与校验

如需修改源代码，请使用 Node.js 22.13 或更高版本，先在根目录运行 `npm install`，再运行 `npm run dev`。重新生成提交用静态成品时运行 `npm run build:static`。

校验命令：`npm run lint`、`npm run typecheck`、`npm test`。

正式公开前仍需由校方确认人物肖像、视频与品牌素材的发布授权，并完成最终内容审校。世界地图底图由阿尔·麦克唐纳创作、弗里茨·莱克沙斯编辑，按“知识共享 署名—相同方式共享 3.0”许可使用。

## 公开仓库安全说明

公开版本不提交 `node_modules`、构建缓存、环境变量、静态生成目录或复盘截图。部署流程只在 `main` 分支更新时执行，并使用最小权限与固定提交版本的 GitHub 官方 Actions。
